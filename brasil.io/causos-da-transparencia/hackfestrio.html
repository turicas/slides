## Problemas no Uso de Dados Públicos - **Formato** (qualidade técnica) - Exemplo: Boletim de balneabilidade em PDF (INEMA/BA) - **Dispersão** - Exemplo: Filiações partidárias: baixar 945 ZIPs (TSE) - **Quantidade** de dados - Exemplo: 18 milhões de sócios - não abre no Excel (RFB) - **Domínio** da área - Exemplo: termos técnicos das eleições (TSE)
## O que podemos fazer? - Libertação de dados - Scripts de scraping e conversão de dados - Plataformas de dados abertos - Educação tecnológica - Interdisciplinaridade com outras áreas - Educação para a cidadania - Pressão através da LAI + mídia - Mudar as leis? - Projetos de inovação cívica
