## QRCode usage - Sharing short URLs - Payments - Tickets - Inventory - ... but it can store up to ~3kB of data!
## Hello, DjangoConUS! ``` import qrcode from qrcode.image.styledpil import StyledPilImage qr = qrcode.QRCode( version=1, error_correction=qrcode.constants.ERROR_CORRECT_L, box_size=10, border=4, ) qr.add_data(b"Hello, DjangoConUS!") qr.make(fit=True) qr.make_image().save("qrcode.png") ``` ![](images/hello-djangoconus.png)
